class bedrock_packs._RpSoundsJsonPart(sounds_json)¶

Bases: abc.ABC

A part of sounds.json file. An abstract base class for 5 different types of the objects contained in sounds.json.

class bedrock_packs._PermanentJsonWalkerContainer(json)¶

Bases: abc.ABC

Holds reference to JsonWalker which can’t be changed. An abstract base class for classes that represent some unmuteable part of JSON file.

property json¶

A JsonWalker with the content of this JSON file.
