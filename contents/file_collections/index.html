File Collections¶

class bedrock_packs._McFileCollection(*, path=None, pack=None)¶

Bases: Generic[bedrock_packs.MCPACK, bedrock_packs.MCFILE], abc.ABC

Collection of files that contain objects of a certain type ( _McFile collection).

__getitem__(key)¶

Get a file that belongs to this collection by using its path, identifier, index or a combination of these properties in a slice object [<path>:<identifier>:<index>].

  • path: Union[Path, str] - the path to the file

  • identifier: str - the identifier used by Minecraft to identify the the object in the file.

  • path: index - in case of multiple files that have the same identifier (which shouldn’t happen in a valid pack), this property can be used to select one of them.

Parameters

key (Union[str, slice]) – the key with combination of properties of an _McFile (path, identifier and index) that let you identify an object that you want to access from the collection.

Return type

~MCFILE

__iter__()¶

Returns an iterator which yields files contained in this collection using their keys. If a key matches to multiple files than only the first one is returned. If you want to iterate over every file use the “objects” property.

class bedrock_packs._McFileCollectionQuery(collections_type, collections)¶

Bases: Generic[bedrock_packs.MCFILE]

Groups multiple file collections (from multiple packs). Used by Project to provide methods for finding McFile in groups of McFileCollection objects that belong to that project.

__getitem__(key)¶

Get a file that belongs to one of the collections grouped in this object.

Parameters

key (Union[str, slice]) – the key with combination of properties of an _McFile (path, identifier and index) that let you identify an object that you want to access from the collection.

Return type

~MCFILE

__iter__()¶

Returns an iterator which yields McFiles for each key from keys() method. If multiple McFiles use the same key only the first one is returned.

Return type

Iterator[~MCFILE]

keys()¶

Returns a list of the identifiers that can be used to access _McFile objects from the collections that belong to this _McFileCollectionQuery.

Return type

Tuple[str, …]
